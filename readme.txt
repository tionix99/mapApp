Тестовое проект.
Используемые фреймворки/библиотеки:
	
	для приложения: 
		- api-maps.yandex v2.1
		- jquery-3.3.1.min
		- jquery-ui.min
		- react.production.min v16.7.0
		- react-dom.production.min v16.7.0
	
	для сборки:
		- babel v6.26
		- webpack v4.28.3
	
	для разработки:
		- webpack-dev-server v3.1.14

Работа с проектом:
	- установка пакетов указанных в package.json: npm_i.bat
	- запуск webpack-dev-server: start_dev_serv.bat
	- создание минифициованной сборки: start_prod_min.bat (выходная папка public)	
	- запуск минифицированной сборки: index_min.html
	
	
Приложение определяет истинное (по геодезической линии) и маршрутное (по данным ymaps.multiRouter.MultiRoute) 
расстояние между несколькими указанными географическими метками.

Элементы управления:
	- Кнопка "автозум" включает/выключает автоматическое центрирование и зуммирование карты при изменении координат географической метки.
	- Кнопка "маршрут" прокладывает маршрут между указанными георафическими метками.
	- Выпадающее меню "формат" предоставляет выбор формата указываемых координат - градусы-десятые градуса или градусы/минуты
	- Выпадающее меню "шаг" позволяет выбрать шаг изменения координаты в диапазоне 1/5/10/15/20 единиц в окне ввода
	- Координанаты "name": 
		- Широта [][] Долгота [][] - окно ввода координат географической метки "name"
		- Кнопка "Х" - удаление с карты географической метки "name"
	- Клик левой кнопкой по карте - устанавливает метку
	- Клик правой кнопкой по метке - удаляет метку
	- Перетаскивание метки ...гм перетаскивает метку
	- Перетаскивание границы окна приложения меняет размер окна
